## ✨ Sobre o projeto

Este é o meu portfólio pessoal, desenvolvido com **HTML5, CSS3 e JavaScript puro**, sem frameworks externos. Ele apresenta:

- 💻 Design **dark mode** com gradientes neon (roxo e azul)
- 🎨 Animações fluidas: typewriter, partículas, scroll reveal, cards 3D
- 📱 Totalmente **responsivo** (mobile e desktop)
- ⚡ Cursor personalizado e efeitos de ripple
- 🧩 Seções: Sobre, Habilidades, Projeto Integrador, Experiências e Contato
